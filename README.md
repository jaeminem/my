# 📅 주간 업무 일정 캘린더

AI 기반 자동 일정 추출 기능을 갖춘 기업용 주간 업무 일정 관리 웹앱입니다.

---

## ✅ 구현된 기능

### 핵심 캘린더
- 주차별 업무 일정 테이블 (월~금 5열 × 7개 구분 행)
- 이전/다음 주 이동, 오늘 주차 바로가기
- 주차 라벨 자동 계산 (YYYY년 MM월 N주차)
- 오늘 날짜 열 자동 하이라이트

### 일정 구분 (7종)
| 구분 | 색상 | 설명 |
|------|------|------|
| 고정 일정 | 파랑 | 매주 반복 회의/업무 |
| 신규 일정 | 초록 | 비정기 신규 일정 |
| 외부기관 현장점검 | 보라 | 감사, 시찰, 점검 |
| 한마음 활동 | 핑크 | 행사, 팀빌딩 |
| 인사발령 | 주황 | 승진, 전보, 채용 |
| 출장/연차 | 진보라 | 출장, 연차, 반차 |
| 기타현황 | 회색 | 기타 |

### AI 자동 일정 추출
- **이미지(JPG/PNG)**: Claude Vision API로 텍스트/일정 인식
- **PDF**: PDF.js로 텍스트 추출 후 Claude API 분석
- **엑셀(XLSX/XLS)**: SheetJS로 파싱 후 Claude API 분류
- 드래그앤드롭 파일 업로드 지원
- 추출된 일정 미리보기 및 수정 기능
- 확인 후 캘린더에 일괄 추가

### 일정 관리 (CRUD)
- 셀 내 [+ 추가] 버튼으로 직접 입력
- 일정 클릭 → 상세 보기 → 수정/삭제
- 구분, 요일, 시간, 장소, 참석인원, 비고 관리

### 내보내기 & 인쇄
- **엑셀 내보내기**: 주간캘린더 + 일정목록 2개 시트
- **인쇄**: 브라우저 인쇄 최적화 CSS 적용

### 설정
- Claude API Key 로컬 저장 (서버 미전송)
- 모델 선택: claude-opus-4-5 / claude-sonnet-4-5 / claude-haiku-3-5
- 기관명, 작성자 설정
- 전체 데이터 초기화

## 🚀 사용 방법

### 1. Claude API Key 설정
1. 우상단 ⚙️ 설정 버튼 클릭
2. Claude API Key 입력 (Anthropic Console에서 발급)
3. 저장

### 2. AI 자동 일정 추출
1. **AI 분석 업로드** 버튼 클릭
2. 이미지/PDF/엑셀 파일 업로드 (드래그앤드롭 가능)
3. **AI 분석 시작** 버튼 클릭
4. 추출된 일정 확인 및 수정
5. **캘린더에 추가** 버튼 클릭

### 3. 수동 일정 추가
- 상단 **+ 일정 추가** 버튼 또는
- 캘린더 셀의 **+ 추가** 버튼 클릭

### 단축키
| 키 | 동작 |
|-----|------|
| `Alt + ←` | 이전 주 |
| `Alt + →` | 다음 주 |
| `N` | 일정 추가 모달 |
| `Esc` | 모달 닫기 |

## 🗂️ 파일 구조

```
index.html          메인 페이지
css/
  style.css         전체 스타일 (기업용 디자인)
js/
  utils.js          유틸리티 함수 (날짜, 포맷, 상수)
  api.js            Table REST API + Claude API 연동
  calendar.js       캘린더 렌더링 및 CRUD 로직
  upload.js         파일 업로드 & AI 분석 모듈
  export.js         엑셀 내보내기 & 인쇄
  app.js            앱 초기화 & 설정 모듈
```

## 📊 데이터 모델 (schedules 테이블)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID (자동 생성) |
| year | number | 연도 |
| week_number | number | ISO 주차 번호 |
| week_label | text | 주차 라벨 |
| day_of_week | text | 요일 (mon~fri) |
| date | text | 날짜 (YYYY-MM-DD) |
| category | text | 구분 코드 |
| title | text | 일정 제목 |
| time_start | text | 시작 시간 |
| time_end | text | 종료 시간 |
| location | text | 장소 |
| participants | text | 참석 인원 |
| note | text | 비고 |
| source | text | 입력 출처 (manual/ai_image/ai_pdf/ai_excel) |

## ⚠️ 주의사항

- Claude API Key는 **로컬 스토리지에만 저장**됩니다 (서버 전송 없음)
- PDF.js CDN을 사용하므로 인터넷 연결 필요
- 이미지 AI 분석은 Claude API 사용량에 따라 비용 발생
